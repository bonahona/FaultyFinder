# Faulty Finder
This is a helper tool for Unity3D, designed to find prefabs that either have references to scripts that are no longer
present in the project or prefabs that are lacking required components as defined in other scripts by
``[RequireComponent(typeof(Collider))]``

## Getting started
First you need a cope of the software. It can be downloaded from the following sources.
* <https://github.com/bonahona/faultyfinder>
